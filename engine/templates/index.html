Last days of Earth

Thank you for playing.

Turns out making an adventure game requires a lot of content.

More than is possible to make for a game jam!

The full game was planned to include a number of cooperative puzzles as you attempt to save the world from climate disaster, with Donald Trump as the final puzzle.

The "spectrum" theme was to be approached using the colour spectrum to represent the rising temperature in the game rooms as the timer expired.

Never mind. Live and learn! We hope you enjoyed this short demo of what we were trying to do :)
